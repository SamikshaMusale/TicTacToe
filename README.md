## **How It Works**

1. **Player X Starts:** The first turn always belongs to X.  
2. **Players Alternate:** Tap a square to place your mark.  
3. **Check for Winner:** The game checks all winning combinations after each move.  
4. **Win or Draw:** The board displays the result instantly.  
5. **Play Again:** Hit restart to begin a fresh match.
